# Vector — Корпоративные квесты

Лендинг для проекта корпоративных квестов **VECTOR**. Полностью адаптивный одностраничный сайт, сверстанный на чистом стеке без фреймворков.

## Функциональность

- Адаптивная вёрстка (mobile 320px / tablet 768px / desktop 1440px)
- Бургер-меню с анимацией и выдвижным дровером
- Горизонтальный drag-scroll галереи и блока отзывов
- Фильтрация форматов квестов по категориям 
- FAQ-аккордеон с плавным раскрытием
- Модальное окно заявки
- Плавный скролл к форме по CTA-кнопкам
- SVG-логотип с анимированной стрелкой 
- SVG-угол с анимацией отрисовки в футере
- Яркий оранжевый футер с контактными ссылками

## Технологии

- HTML5 (семантическая разметка)
- CSS3 (Vanilla CSS, без фреймворков)
- JavaScript (Vanilla JS, без библиотек)
- БЭМ — методология именования классов
- CSS-переменные (`--token`) для дизайн-системы
- SVG — векторная графика и анимации

## Дизайн-система

Все токены хранятся в `css/tokens.css`. Чтобы поменять акцентный цвет на всём сайте — достаточно изменить одну переменную:

```css
:root {
    --color-primary: #e85d1a;   /* акцентный оранжевый */
    --color-bg-base: #0a0905;   /* фон страницы */
    --font-display: 'Zvezda NHZDN', serif;
    --font-body: 'FindSans Pro', sans-serif;
}
```

| Брейкпоинт | Устройство | Поведение |

| ≤ 375px | Узкий мобайл | Уменьшенные заголовки и фото |
